# 🚀 Pixel Perfect Predictor | GDG Template

Welcome to the GDG Constructor University deployment template! Fork this repo to instantly turn your AI model into a live, branded web application.

## 🛠️ How to Deploy Your AI

1. **Fork it:** Click the "Fork" button in the top right corner to copy this code to your own GitHub account.
2. **Export:** In your Teachable Machine browser tab, click "Export Model", select **TensorFlow.js**, and download the `.zip` file.
3. **Upload:** Unzip that download. Drag and drop the `model.json`, `metadata.json`, and `weights.bin` files directly into this GitHub repository to overwrite the placeholders.
4. **Deploy:** Go to your repository **Settings** -> **Pages**. Under "Build and deployment", change the Source branch to `main` and hit Save.
5. Wait 60 seconds, refresh the page, and your AI is live for the world to see! 🔥
